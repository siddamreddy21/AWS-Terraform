Installation Terraform CLI(In windows user Powershell with Administarator)

1) Need to install choco CLI 
2) Need to install Terraform cli : cmd-> choco install terraform
3) Verify installation : cmd -> terraform -help  (or) cmd -> terraform -help plan
4) Need to install HashiCorp plugin in Visual Code
5) Need to install AWS CLI in local
6) Need to download the .msi file from the link : https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html
7) to check the if aws cli is installed or not cmd: aws --version
